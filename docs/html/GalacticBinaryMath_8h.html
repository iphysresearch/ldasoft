◆ cholesky_decomp()

void cholesky_decomp ( double **  A,
double **  L,
int  N 
)

Wrapper to GSL Cholesky decomposition routine.

Copies matrices into gsl_matrix structures and then fills the lower half of \(L\) with the result. GSL returns the original matrix in the upper half of \(L\). We replace the upper half of \(L\) with all 0s. Contents of \(A\) are unaltered.

Parameters
[in]Nsize of matrix
[in]A\(N\times N\) matrix \(A\)
[out]LCholesky decomposition of \(A = LL^{-1}\)

double snr ( struct Source *  source,
struct Noise *  noise 
)

Signal to noise ratio.

Calls nwip() to compute inner product of waveform with itself summed over all frequencies and data channels I. If 4-link data, use single TDI channel I = X. If 6-link, use I = {A, E}

Parameters
sourcestructure containing source parameters and waveform \(h\)
noisestructure containing noise model \(S_n\)
Returns
\(\rho = \sqrt{\sum_I (h_I|h_I)} \)
